# BOQ Extractor Pro v20.0 - Clean Edition

Simple, working BOQ extraction with material review.

## What's Fixed

- **Material Merging**: Auto-detects when "Graphite bronze" merges with description
- **Simple Review**: Clean inline editing, no complex state management
- **Reliable Export**: Tested Excel/CSV generation

## Deploy

1. Push to GitHub
2. Deploy to [share.streamlit.io](https://share.streamlit.io)
3. Main file: `app.py`

## Usage

1. Upload PDF
2. Adjust crop area
3. Click "Extract & Review Materials"
4. Fix any merged materials
5. Finalize and download Excel
